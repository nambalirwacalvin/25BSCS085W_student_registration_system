•	Brief description of the system
The Student Registration System is a web-based application that allows educational institutions to efficiently manage student enrollments. It provides functionality for adding new students, viewing registered student records, updating student information, and deleting entries. The system captures essential details such as full name, number, course enrolled. It is designed to reduce paperwork, eliminate duplicate data entry.
